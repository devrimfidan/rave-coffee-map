# ☕ Rave Coffee Subscription Journey Map

Welcome to my **Rave Coffee Subscription Journey Map**! This is an interactive, fan-made geospatial dashboard designed to visually document and track all of the delicious coffees I have received so far as a happy member of the **Rave Coffee Subscription**.

## 🛠️ The Tech Stack

This project was built from scratch and progressively polished using standard, high-performance web APIs and modern AI assistance:
1. **Interactive Cartography**: Powered by **Leaflet.js** and styled with CartoDB's Dark Matter tileset to match Rave's sleek, high-contrast black-and-yellow colorway.
2. **AI Pair Programming**: Crafted, refactored, and iterated collaboratively using:
   * **Google Antigravity SDK** for autonomous file orchestration.
   * **Gemini 1.5 Flash** for content generation, descriptions, and dataset completion.
   * **Claude 3.5 Sonnet** for visual polish, spring transitions, dynamic clustering, and styling.
3. **Pure Web Technologies**: Pure HTML5, CSS3, and modern Vanilla JS (ES6) with zero heavy build steps.
